\begin{frame}

\vspace{\fill}

\begin{center}

An \textbf{arithmetic} is a numeric data type \\ and a collection of functions
over the data type.

\end{center}

\vspace{\fill}

\end{frame}


\begin{frame}

\vspace{\fill}

\begin{center}

A \textbf{digital arithmetic} is an approximation of the above.

\end{center}

\vspace{\fill}

\end{frame}
